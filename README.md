#### Используемые технологии:
+ JavaScript
+ React
+ React Hooks
+ React Router
+ MobX, Сontext (в отдельных ветках)
+ SCSS
+ HTML5/CSS3

GitHub Pages: [English Dictionary](https://8alex.github.io/flashcards_app/)

Запуск проекта:
```
git clone https://github.com/8Alex/flashcards_app.git 
cd flashcards_app 
npm install  
npm start
```
